SIRI

This is one of the main projects created by Apple Inc, ‘Siri.’ This concept is created by using the AI technology and this was implemented to the apple iPhone 4S in 2011. Later on, the voice assistant was implemented to the rest of the apple devices. Using machine learning technology allows Siri to play multiple functions. In fact, it also allows the user to change Siri’s voice making comfortable for the user to use. Having this facility in user’s phone has change their lives as it does the action whenever a user speaks up. Saying ‘Hey Siri!’ allows to activate Siri and then it listens to what user says.

As Apple is renowned for its quality assurance and thinking ahead compared to other tech companies, in 2016 Apple has enhanced the feature giving Siri to access third-party apps such as messaging apps, payment apps, internet calling apps but up to limited apps as Apple is also considered to be the best security provider for its customers. Having these features has led a major change in humans’ lives.
